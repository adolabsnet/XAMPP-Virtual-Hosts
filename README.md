# XAMPP Virtual Hosts
Create and manage Virtual Hosts for XAMPP.

## Features
- User Friendly GUI to manage all your Virtuals Hosts
    - View Virtual Hosts
    - Add Virtual Hosts
    - Delete Virtual Hosts
- Automatic backups

## Requirements
- Microsoft .NET Framwork 4.7.2
